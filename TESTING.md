## ✅ テストチェックリスト

### A. トップ画面（ツール選択）
- [ ] 3つのツールカード（漫画・スライド・デザイン）が表示されている
- [ ] カードがデスクトップ/タブレット/モバイルで見やすく並んでいる
- [ ] ホバー時のアニメーションが自然
- [ ] タイポグラフィ（Outfit + Noto Sans JP）が正しく表示されている
- [ ] Amberアクセント色が一貫している（シアン/水色が残っていないか）

### B. AI漫画クリエイター
- [ ] モード選択（Q&A / フォーム）が直感的
- [ ] パンくずナビゲーション "Studio › AI漫画クリエイター" が表示
- [ ] Q&A画面のプログレスバーが明瞭
- [ ] 質問カードの余白・視認性
- [ ] 「戻る」で前の画面に正しく戻れる

### C. AIデザインクリエイター（エディタ）
- [ ] ツールバー（左）のアイコンが認識できる
- [ ] レイヤーパネル・プロパティパネル（右）の日本語化
- [ ] 選択ボックス・ハンドルの色が新Amberパレット
- [ ] 書き出しドロップダウン（PDF/PNG/SVG/IDML）が開く
- [ ] Affinity/Figma と比較した操作感

### D. レスポンシブ
- [ ] スマホ幅（375px）でレイアウトが崩れない
- [ ] フィードバックボタンが他の要素と重ならない
- [ ] 著作権バナーが邪魔にならない

### E. インタラクション
- [ ] ボタンホバー時のフィードバックが適切
- [ ] クリック時の反応（scale 0.98のpress effect）
- [ ] タッチ/マウス両方で違和感がないか

### F. アクセシビリティ
- [ ] キーボード操作（Tab移動）でフォーカスが見える
- [ ] テキストコントラストが読める
- [ ] スクリーンリーダー対応（aria-label等）

## 🙏 フィードバックの型

以下の構造で書いていただけると助かります:

```
【現状】 スクリーンショット + 観察した事実
【問題】 なぜ改善すべきか
【提案】 どう変えるべきか（参考URL/画像あれば）
【優先度】 Critical / High / Medium / Low
```
